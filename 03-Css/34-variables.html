34 Variables

The var() function is used to insert the value of a CSS variable.

:root { --color-primary: #4fc3f7; --color-secondary: #ffb74d; --color-text: #e0e0e0; --spacing-medium: 1rem; --font-family-base: 'Poppins', sans-serif; }

🔹 Variables declaradas en :root para uso global.

.elemento { color: var(--color-primary); padding: var(--spacing-medium); font-family: var(--font-family-base); }
